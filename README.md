# Brnach: `project/CRUD-on-expenses`

This brnach Contains CRUD operations for managing expenses. Each expense item include an ID, description, and amount.

- **GET** fetches all items and can also process a query to search for a specific item.
- **POST** creates a new item using smart ID handling.
- **PUT** updates one or more parts of an item.
- **DELETE** removes an item from the dictionary database using its ID.
